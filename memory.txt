== КАК РАБОТАЕТ ==

Бот (bothost.ru):
1. Пользователь пишет /start
2. Бот просит ввести номер телефона или номер ВУ
3. Бот отправляет запрос на API: /api/driver/verify?identifier=XXX
4. API проверяет в Yandex Fleet API (через Cloudflare Worker)
5. Если найден - открывается доступ к Mini App
6. Если нет - "Номер не найден в базе парка"

Mini App (zolotoybot.ru):
- Главная: активный заказ (откуда→куда, цена, статус)
- История: завершённые заказы
- Статистика: заработок за день
- Настройки: водитель, авто, оплата (нал/безнал)
- Поддержка: FAQ + ссылка на чат

API (zolotoybot.ru):
- /api/auth/* - авторизация
- /api/driver/* - водитель (info, stats, payment, car, verify, check_phone)
- /api/orders/* - заказы (active, history, cancel, complete)
- /api/webhooks/yandex - входящие webhooks от Yandex
